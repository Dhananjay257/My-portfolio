Projects

E-Commerce

The E-Commerce project is a Spring-based application simplifying online shopping with 10 APIs efficient management of sellers, products, and payments. Leveraging Spring Boot and Hibernate ORM, it ensures seamless data handling with MySQL integration.!

E-Commerce
  • This is Spring-based shopping application with 10 APIs that manage sellers, products, customers, carts and cards, which simplifies the shopping experience.
  • Utilized Hibernate ORM for efficient data management and integrated with MySQL database for data persistence and DTOS to ensure privacy.
  • It enables user to add their card details,add items to cart and eventually checkout.
  • It enables customer to register themselves and place an order and to fetch product list by specifying category or price along with the category.
  • The application has a the functionality of sending real time emails that enables users to get order confirmation and shipping updates.
  • Integrated Swagger for documenting APIs and providing an interactive user experience.
  • Tech Stack used: Postman, Intellij, SpringBoot, MySQL, JPA/Hibernate

Library Management Systems

The Library Management System streamlines book transactions for students with 7 APIs, managing accounts and author records efficiently. Utilizing Spring Boot and Hibernate ORM, it reflects my expertise in Java, Spring Boot, and MySQL, and dedication to efficient solutions.

Library Management System
  • It is dynamic Spring application with 7 APIs simplifies book transactions for students and makes library accounts, books, and authors easy to manage.
  • Utilized Hibernate ORM for efficient data management and integrated with MySQL database for data persistence and DTOS to ensure privacy.
  • Features include automatic card issuance, streamlined book issuing returning and transaction recording.
  • The application has a the functionality of sending real time emails when the student issues the book.
  • Integrated Swagger for documenting APIs and providing an interactive user experience.
  • Tech Stack used: Postman, Intellij, SpringBoot, MySQL, JPA/Hibernate
